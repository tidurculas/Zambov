# Zambov photo bundle

This package is ready to upload to GitHub.

## Included
- index.html
- style.css
- script.js
- assets/audio.mp3
- assets/photo-01.jpg ... assets/photo-24.jpg

## GitHub update
1. Open your GitHub repo.
2. Upload and replace all files from this package.
3. Commit the changes.
4. Wait for GitHub Pages to redeploy.
5. Hard refresh the site.

## Notes
- Slideshow changes every 3 seconds.
- It loops continuously while the poem is being read.
- On the final line, the slideshow freezes on whichever photo is currently visible.
- UI fades away on the last line.
- Audio is already bundled as assets/audio.mp3.
